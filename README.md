# User-centric
Insight into the User behind the Text: \\An LLM Powered User-Centric Text Opinion Analysis Framework

This project is a user-centric opinion analysis framework that utilizes Bayesian Prompt Optimization (BPO) to align user profiles with the reasoning processes of Large Language Models (LLMs).

Project Structure
The core code of the project is distributed across the following two main modules:

1. /BPO (Bayesian Prompt Optimization)
This directory contains all the core algorithmic logic for the prompt optimization phase.


Gaussian Process (GP) Surrogate Model: Constructs a probabilistic mapping of the objective function (downstream F1 score) to achieve efficient black-box optimization.

Acquisition Function (EI): Utilizes Expected Improvement to balance exploration and exploitation within the semantic space.

Optimization Iteration: Supports closed-loop optimization for soft prompts generated by various white-box models.

2. /agent (Decision Agent)
This directory contains the implementation of the decision agent, which is responsible for the final opinion analysis task.

Black-Box Model Integration: Integrates mainstream black-box LLMs (e.g., GPT series) by default, performing inference via API calls.

Role Interaction Logic: Defines the collaborative workflow among the Profile Prompter, Profile Creator, and the decision module.

Note. The model in the agent file can be replaced with a model trained using BPO or any large open-source model.
